# 📚 Student Management System

## Quick Start
```bash
cd sms
pip install flask openpyxl
python app.py
```
Open: http://localhost:5000

## Login
- Email: admin@sms.com

## Features
- Login/Signup with password hashing
- Student CRUD (Add, Edit, Delete)
- Attendance tracking
- Grade management
- Fee records
- Dashboard stats & reports
- All data stored in students_db.xlsx
